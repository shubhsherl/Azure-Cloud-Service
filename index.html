Product Overview

The Azure Linux AMI is a supported and maintained Linux image provided by MS Services for use on Azure Elastic Compute Cloud. It is designed to provide a stable, secure, and high performance execution environment for applications running on Azure. It also includes packages that enable easy integration with MS Services, including launch configuration tools and many popular AWS libraries and tools. Azure Web Services provides ongoing security and maintenance updates to all instances running the Azure Linux ALI. The Azure Linux ALI is provided at no additional charge to Azure users.
